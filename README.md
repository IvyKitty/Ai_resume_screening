*AI Resume Screening & Candidate Ranking System*

🚀 *Project Overview*

This project is an AI-powered recruitment tool developed during my internship at LaunchED Global. It streamlines the hiring process by using Natural Language Processing (NLP) to analyze resumes against a specific job description and rank candidates based on their skill relevance.

Live Demo: https://airesumescreening-hvwehj7dw2bv739yuvyue5.streamlit.app/

🛠️ *Technical Stack*

-Frontend: Streamlit (Web UI)

-Backend: Python

-NLP Engine: Scikit-learn (TF-IDF Vectorization & Cosine Similarity)

-PDF Parsing: PyPDF2 / pypdf

-Data Handling: Pandas & NumPy

-Deployment: Streamlit Cloud

✨ *Key Features*

-Multi-PDF Support: Upload multiple resumes simultaneously for bulk processing.

-Instant Ranking: Uses TF-IDF (Term Frequency-Inverse Document Frequency) to extract key features from text.

-Similarity Scoring: Calculates a match percentage using Cosine Similarity between the job description and each resume.

-Interactive UI: Displays results in a clean, sorted table for easy candidate shortlisting.
